API Key

Head over to the API Keys section on your account page, or:

  • Click on your profile/account icon
  • Click 'Account'
  • Click on the 'Extras' dropdown, select 'API Keys'
  • Either create a new key, or use an existing one

Audience ID

  • Go to the 'Audience' page
  • Select an audience from the 'Current audience' dropdown
  • Click on the 'Manage Audience' dropdown on the right, select 'Settings'
  • Click on the 'Settings' dropdown, select 'Audience name and defaults'
